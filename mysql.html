Reset the MySQL root password on macOS

These notes are from 2015 and assume a /usr/local/mysql install. Newer MySQL versions use ALTER USER instead of updating mysql.user directly — adjust to your version.

Steps

  1. Stop the mysqld server: sudo /usr/local/mysql/support-files/mysql.server stop
  2. Run the server in safe mode with privilege bypass: sudo mysqld_safe --skip-grant-tables
  3. In a new window, connect to the database: mysql -u root
  4. Set a new password: UPDATE mysql.user SET Password=PASSWORD('password') WHERE User='root';
  5. Flush the permissions: FLUSH PRIVILEGES;
  6. Quit: \q
  7. Stop the safe-mode server by pressing Control + Z
  8. Start your regular server again: sudo /usr/local/mysql/support-files/mysql.server start
  9. The new password should work now.
